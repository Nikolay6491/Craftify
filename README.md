В приложении будут реализовываться функции:
- регистрация и аутентификация;
- лайки;
- коментарии;
- возможность поделиться постом;
- отображение количества просмотров;
- геолокация;
- мультимедиа (возможность просмотра фото/видео).

Библиотеки:
- LiveData;
- Android DataBinding;
- AndroidX;
- Hilt;
- ViewModel;
- Retrofit;
- Dagger;
- Room;
- Glide;
- Android Testing;
- Kotlin Coroutines;
- ImagePicker;
- MapKit;
- Material Design;
- MVVM;
- Flow;
- Paging.
